1. Import Tkinter

Copy

import tkinter as tk
Tkinter is the standard GUI toolkit in Python. We use it to create GUI windows, buttons, and other widgets.

2. Function to Handle Button Clicks
python
Copy
Edit
def click(event):
    text = event.widget.cget("text")
This function is triggered when a button is clicked.

event.widget.cget("text") gets the text of the button that was clicked.

4. Create Main Window
python
Copy
Edit
root = tk.Tk()
root.title("Calculator")
root.geometry("300x400")
Creates the main application window with a title and fixed size.

5. Create Input Field
python
Copy
Edit
entry = tk.Entry(root, font="Arial 20", borderwidth=4, relief=tk.RIDGE)
entry.pack(fill=tk.BOTH, ipadx=8, pady=10)
tk.Entry creates a single-line input field.

fill=tk.BOTH allows it to expand horizontally.

font makes it look nicer.

6. Define Buttons in a Grid
python
Copy
Edit
buttons = [
    ['7', '8', '9', '/'],
    ['4', '5', '6', '*'],
    ['1', '2', '3', '-'],
    ['C', '0', '=', '+']
]
This list defines the calculator buttons row by row.

7. Create Frames and Buttons
python
Copy
Edit
for row in buttons:
    frame = tk.Frame(root)
    frame.pack(expand=True, fill="both")
Each row of buttons is placed inside a frame for better layout control.

python
Copy
Edit
    for btn in row:
        button = tk.Button(frame, text=btn, font="Arial 18", relief=tk.GROOVE, border=0)
        button.pack(side=tk.LEFT, expand=True, fill="both", padx=2, pady=2)
        button.bind("<Button-1>", click)
Creates each button, styles it, and binds the click function to it.

8. Start the GUI Loop
python
Copy
Edit
root.mainloop()
This line tells Python to keep the window open and wait for user interaction (event loop).
